Nach Installation von MongoDB

-> in root Verzeichnis einen 'data' Ordner anlegen und darin einen 'db' Ordner
-> zu Verzeichnis navigieren in dem MongoDB installiert wurde.
-> -> mongod.exe ausführen
-> npm init --yes um package.json File zu generieren (in Projektordner)
-> mongoose installieren: npm install mongoose --save

(MongoDB kann dann mit 'mongo' Kommando gestartet werden)

MongoDB Connection herstellen
(Client -> Server[NODE] mit mongoose -> MongoDB)
-> connection.js file
-> node Verzeichnis/connection.js um Verbindung zu testen

Datenbanken/ Collections können in MongoDB Compass angelegt werden (connection string zu DB wird automatisch beim Öffnen erkannt. Port 27017 )
